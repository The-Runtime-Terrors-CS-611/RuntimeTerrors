Technological Choices Narrative

Forged In Fire will use a NoSQL database, and a multitenant database system. It is a web application and there is not much that will be contained in the database: inventory, relationship status, save files etc. There is not a need for each user to have their own dedicated database. The delivery platform we will develop for is the desktop browser. The goal for the game is to feel old school and being browser-based will achieve this perfectly. We will use a cloud server for ease of use, low initial costs, ease of scalability, etc. The main open-source tool we will use is the Bootstrap framework for streamlining our web applicationâ€™s CSS styling.
